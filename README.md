# Wedding Invitation SaaS (Laravel)

Project ini merupakan aplikasi berbasis web dengan konsep SaaS (Software as a Service) yang digunakan untuk membuat undangan pernikahan digital. Saya melakukan ini, bertujuan untuk belajar cara menjadi web dev.

#Lama pengembangan
Saya belajar laravel baru sekitar 2 - 3 mingguan dengan rata rata 2jam sehari selama 4-5 hari tiap minggunya, namu itu saya hentikan sesaat untuk mempelajari Python dan Django, setelah melihat lowongan pekerajaan yang memerlukan pemahaman pyhton

## Fitur
- Halaman undangan digital
- Routing dan controller berbasis Laravel
- Tampilan menggunakan Blade Template
- Login basis token
